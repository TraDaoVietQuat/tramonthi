KHỞI ĐỘNG NHANH

  ┌─ KHOI_DONG.bat  (dành cho lập trình viên) ───────────────────┐
  │  Mở terminal hiển thị log, tự động mở trình duyệt           │
  │  Tự phát hiện chế độ:                                        │
  │    • Có bản build sẵn → production (~1–2 giây)              │
  │    • Không có build   → dev mode (hot reload)               │
  └───────────────────────────────────────────────────────────────┘

  Để đóng gói chia sẻ: Double-click DONG_GOI.bat
    → Tự build + nén thành TramOnThi-FullPackage.zip

HƯỚNG DẪN HỌC SINH
────────────────────
  Bước 1 – Đăng ký / Đăng nhập
    1. Vào http://localhost:3000/register
    2. Điền họ tên, email, mật khẩu (tối thiểu 6 ký tự)
    3. Nhấn "Tạo tài khoản" → chuyển vào Dashboard

  Bước 2 – Chọn khóa học
    1. Vào Dashboard: http://localhost:3000/dashboard
    2. Xem danh sách 16 khóa học (Lớp 10 / 11 / 12 / Ôn thi ĐH)
    3. Nhấn "Bắt đầu học" hoặc "Tiếp tục" trên card khóa học

  Bước 3 – Học bài
    1. Đọc nội dung bài học (có bảng, công thức, ví dụ)
    2. Nhấn "Đánh dấu hoàn thành" khi xong mỗi bài
    3. Dùng sidebar bên phải để điều hướng giữa các bài

  Bước 4 – Làm bài kiểm tra
    1. Sau mỗi module, nhấn "Làm bài kiểm tra"
    2. Chọn đáp án cho từng câu hỏi trắc nghiệm (4 lựa chọn)
    3. Nhấn "Nộp bài" → xem điểm số, đáp án đúng/sai

  Bước 5 – Khảo sát
    • Pre-test  : hiện lần đầu vào Dashboard
    • Post-test : hiện sau khi hoàn thành khóa học

  Hỏi bài với AI
    • Nhấn nút chat xanh lá góc phải màn hình
    • Gõ câu hỏi về Toán, Lý, Hóa, Anh → AI trả lời ngay
    • Enter để gửi, Shift+Enter để xuống dòng


HƯỚNG DẪN GIẢNG VIÊN
──────────────────────
  Đăng nhập : http://localhost:3000/instructor/login
  Portal     : http://localhost:3000/instructor  (tất cả tính năng tại đây)

  Giao diện gồm sidebar bên trái với 2 mục:

  [Dashboard] — Thống kê & xuất dữ liệu
    • Tổng số học sinh, bài hoàn thành, lượt quiz, điểm trung bình
    • Khảo sát Pre/Post-test: số lượt nộp, tỉ lệ hoàn thành
    • Xuất CSV: kết quả Quiz, tiến độ học bài, khảo sát Pre/Post

  [Quản lý nội dung] — Tạo & chỉnh sửa nội dung
    • Tạo khóa học mới (chọn môn, khối lớp, mô tả)
    • Thêm module vào khóa học
    • Thêm / sửa bài học (soạn nội dung Markdown, upload file PDF/ảnh/Word)
    • Tạo quiz: thêm câu hỏi 4 lựa chọn, chọn đáp án đúng
    • Xuất bản / Thu hồi khóa học (học sinh chỉ thấy khóa đã xuất bản)
    • Xóa module, bài học, câu hỏi, khóa học

  Lưu ý phân quyền
    • Giảng viên KHÔNG vào được /admin
    • Admin KHÔNG vào được /instructor (tách biệt hoàn toàn)


HƯỚNG DẪN QUẢN TRỊ VIÊN
─────────────────────────
  Đăng nhập : http://localhost:3000/admin/login
  Dashboard  : http://localhost:3000/admin
  Giao diện  : Nền trắng chuyên nghiệp (khác biệt với giảng viên)

  Thống kê hệ thống
    • Tổng người dùng / Học sinh / Giảng viên
    • Khóa học (đã xuất bản / tổng số)
    • Lượt làm quiz và điểm trung bình
    • Biểu đồ phân bổ vai trò

  Quản lý người dùng
    • Xem toàn bộ danh sách (tên, email, vai trò, ngày tạo)
    • Đổi vai trò: Học sinh ↔ Giảng viên ↔ Quản trị
    • Xóa tài khoản (yêu cầu xác nhận trước khi xóa)
    • Lưu ý: không thể tự đổi role / xóa chính mình

  Phân quyền truy cập
    • Chỉ ADMIN mới vào được /admin
    • Admin vào /instructor → tự redirect sang /admin
    • Học sinh vào /instructor hoặc /admin → redirect /dashboard

CẤU HÌNH MÔI TRƯỜNG (.env)
────────────────────────────
  File: elearning-app/.env

  DATABASE_URL              → Đường dẫn database SQLite
  JWT_SECRET                → Khóa bí mật JWT
  OPENAI_API_KEY            → API Key OpenAI (chatbot AI)
  NEXT_PUBLIC_RECAPTCHA_SITE_KEY → Site key Google reCAPTCHA v2
  RECAPTCHA_SECRET_KEY      → Secret key Google reCAPTCHA v2


LỆNH THƯỜNG DÙNG
─────────────────
  Khởi động nhanh:
    Double-click KHOI_DONG.bat

  Chế độ phát triển (khi sửa code):
    npm run dev

  Build production (sau khi sửa code xong):
    npm run build

  Tạo dữ liệu:
    npm run db:seed    → Khóa học + tài khoản gốc
    npm run db:demo    → Dữ liệu hoạt động mẫu

  Reset database:
    npx prisma db push --accept-data-loss
    npx prisma db seed
    npm run db:demo

  Xem database trực quan:
    npx prisma studio  →  http://localhost:5555

  Đóng gói để chia sẻ:
    Double-click DONG_GOI.bat  →  TramOnThi-FullPackage.zip
